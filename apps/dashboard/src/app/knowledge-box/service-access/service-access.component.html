The API key can be used on any call to the API. It goes in the X-NUCLIA-SERVICEACCOUNT header and replaces the Authorization header 

curl $$KB_URL$$/upload \
 -X POST \
 -H "content-type: video/mp4"
 -H "x-language: fr"
 -H "X-NUCLIA-SERVICEACCOUNT: Bearer YOUR-API-KEY" \
 -T /path/to/file.mp4
