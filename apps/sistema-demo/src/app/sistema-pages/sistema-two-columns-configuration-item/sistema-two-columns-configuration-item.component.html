Two columns configuration item

Layout component allowing to display two even size columns for configuration pages:

  • The first column contains a title, a description and an optional badge.
  • The second column has free content.

two-columns-configuration-item component allows to build a row in such a two columns layout.

This line doesn't have a top border thanks to its noTopBorder attribute.

This column is free content, you can use it to display whatever you need.
Choice 1 Choice 2 Choice 3

Inputs

badge
Badge displayed next to the title. (optional)
description
Description to be displayed on the left column.
itemTitle
Title to be displayed on the left column.
noTopBorder
When true, prevent the top border to be displayed
