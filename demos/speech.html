This is a demo of the Nuclia Chat widget with the speech feature enabled.

It's more likely to work with Chrome than with other web browsers.

  1. Click the microphone icon to start speaking.
  2. Say "question" to let the widget knows you want to ask a question.
  3. Ask your question (this Knowledge Box contains information about market outlook in 2024, so ask )
  4. Say "search" to trigger the search.
